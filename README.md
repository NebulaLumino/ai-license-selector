# AI License Selector

An AI-powered tool that recommends the best open source license for your project based on your requirements.

## Features

- AI-powered open source license recommendations
- Considers project type, commercial use, distribution, and attribution needs
- Dark gradient UI with violet accent
- Powered by DeepSeek API

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

```env
OPENAI_API_KEY=your-deepseek-api-key
```

Visit http://localhost:3000
